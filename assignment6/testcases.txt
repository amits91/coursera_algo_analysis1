The task (PE6, question 1) asks to count number of values t in the range [−10000,10000] such that there is distinct x and y (in the file) and x+y=t. According to the problem statement both 1 + 2 =3 and 2 + 1 = 3 result in 3 which is counted as a single target value 3. In other words, you don't need to count pairs x and y, you have to count target values t belonging to [−10000,10000]. For example, 3+4, 4+3, 1+6, 6+1, 5+2, 2+5 all express a single target value 7.
